# Laboratuvar Talimat Sistemi

## Dosyalar
- `index.html` — Yönetim paneli (bilgisayarda açın)
- `talimat.html` — QR okutulunca telefonda açılan sayfa
- `pdfs/` — PDF talimatlarınızı buraya yükleyin

## GitHub Pages Kurulum (5 dakika)

1. github.com → New repository → `lab-talimat` (public)
2. Bu 2 HTML dosyasını yükleyin (Add file → Upload files)
3. `pdfs/` klasörü oluşturun, PDF'leri içine yükleyin
4. Settings → Pages → Branch: main → Save
5. Siteniz: `https://KULLANICI_ADINIZ.github.io/lab-talimat`

## Kullanım

1. `index.html`'i tarayıcıda açın
2. ⚙ Ayarlar → GitHub kullanıcı adı + repo adını girin
3. Gruplar ve talimatlar ekleyin, her birine PDF dosya adı yazın
4. QR kodları indirin veya yazdırın
5. Telefon kamerasıyla QR okutun → PDF açılır
